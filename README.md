## jewelry store
היא חנות תכשיטים המציעה מגוון של סוגי תכשיטים - טבעות, עגילים ושרשראות במגוון סוגים וסגנונות

דף הבית:
הדף הראשי שיוצג למשתמש כשהוא נכנס לאתר, בעמוד זה תוצג למשתמש כותרת ותמונה ולחצן למעבר לפריטי המוצרים שבחנות.

דף המוצרים:
